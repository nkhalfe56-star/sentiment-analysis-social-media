# Sentiment Analysis for Social Media

> **Applied AI & Data Science Course Project** | NLP Research

## Overview

An end-to-end NLP pipeline for social media sentiment classification, covering data collection, preprocessing, feature extraction, and model training.

## Key Highlights

- Designed a complete NLP pipeline handling data collection, preprocessing, feature extraction, and model training
- Compared classical ML (Logistic Regression, SVM) with deep learning approaches (LSTM, BERT-based models) using TensorFlow and PyTorch
- **Achieved 87% accuracy on unseen test data**
- Visualized sentiment trends through interactive dashboards (Matplotlib, Seaborn, Plotly), enabling exploratory analysis of public opinion dynamics
- Conducted systematic model comparison and ablation studies demonstrating experimental rigor
